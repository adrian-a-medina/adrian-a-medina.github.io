# Personal Website - Academic Curriculum Vitae
Welcome to my personal website repository! This site serves as a showcase of my professional background, academic achievements, research interests, and other information as I prepare to apply to PhD programs. The website is hosted via GitHub Pages and built from scratch using HTML, CSS, and JavaScript.

## Overview

The main sections of the website include:

1. **Homepage** - A brief introduction about myself, my research interests, and key highlights of my career.
2. **About Me** - Detailed background information including my academic journey, research experiences, and other qualifications.
3. **Research Interests** - A deep dive into my areas of research interest, particularly in the fields of affective neuroscience, developmental psychology, and computational psychiatry.
4. **CV** - An interactive version of my CV, including publications and professional experiences.
5. **Publications** - A list of research publications and conference presentations.
6. **Projects** - A section dedicated to my active projects.
7. **Teaching** - A section about my prior teaching experiences.

## Technologies Used

- **HTML5**: Structuring the content.
- **CSS3**: Styling the website and ensuring responsive design.
- **JavaScript**: Adding interactivity to enhance user experience.
- **GitHub Pages**: Hosting the website online.

## Getting Started

To run the website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/adrian-a-medina/adrian-a-medina.github.io.git
   ```

2. Open the `index.html` file in your preferred browser to view the site.

## Project Goals

- To create a professional, easy-to-navigate personal website.
- Showcase my academic and research journey as a prospective PhD applicant.
- Provide an organized and detailed presentation of my work for potential collaborators, admissions committees, and colleagues.
